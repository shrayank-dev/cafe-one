# ☕ Brew Haven Coffee

A premium, modern coffee shop website designed and developed as part of the **Nexora Studios Portfolio**.

---

## 📖 Overview

Brew Haven Coffee is a modern, responsive landing page created for a fictional premium coffee shop. The website focuses on creating a warm and inviting digital experience while encouraging customers to visit the café.

The project showcases modern UI/UX principles, responsive layouts, smooth animations, and a premium visual identity.

---

## ✨ Features

- Modern responsive design
- Beautiful hero section
- About Us section
- Featured menu
- Signature drinks
- Customer testimonials
- Gallery
- FAQ
- Contact form
- Google Maps integration
- WhatsApp contact button
- Mobile-first layout
- Smooth animations
- SEO-friendly structure
